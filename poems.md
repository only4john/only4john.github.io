# 随便的一些诗

## 接站
我跑着穿过雨和雨的间隙  
我快步穿过人和人的间隙  
我温柔穿过你的间隙  
来抵达你  

## 新药
我所需要的  
是两瓶有待发明的新药  
一瓶用来忘记你  
一瓶用来下决心  
——去忘记你  

## AN END IS MISSING  
Like a function  
which is lame  
like a piece of code  
forgot to return to main()  
"save me, please", it cried  
but the sound  
disappeared in vain  

## 三倍的孤独  
如果没有  
三万公里的远处  
那一颗孤独的通讯卫星  
我无法听见  
你对我倾诉孤独时  
最轻的那声叹息  

## 青春  
当青春枯萎成一个黑名单，一把刺  
我骄傲的对旁人宣布，『美妙』  
我坚定的向自己确认，『无悔』  
可当我梦见当初的自己  
我挤过去碰碰他的手背  
小声说，『快逃』  

## 阅读的退化  
我们都只在阅读里  
等待着感动的来临  
同情的眉毛一落一起  
在鼻子变酸前翻到下个故事  

眼睛寻找下一个焦点  
舌头寻找下一顿晚餐  
从牛羊鱼吃到小龙虾  
吃甲虫都没关系——只要还有啤酒  

我们越来越像一个共同体  
被光纤做脐带彼此连接  
相信少数人的奇遇是我们共同的奇遇  
以为少数人的勇气是我们共同的勇气  

我们只是在阅读里  
等待着感动的来临  
做爱都比阅读积极  
至少还有高潮，还有创造。  

## 让我们拭目以待  
让我们拭目以待  
总有一天  
我会有睿智的额头  
和深深的眼窝  
我言语温和  
举止恬淡  
就像高手百练成绝学  
万钧袭来也只需轻轻侧身  

有一天久违的你打马而来  
夕阳刺眼看不清你的容颜身形  
你应该也老了吧  
你应该也老了吧？  
我猜你会让我领教  
那年一别后  
你所有的历练和修行  

没想到你只是打马走过我身边  
哼唱着年轻时我们熟悉的曲调  
一抬头我发现时间只是一个伪命题  
你就这样把我打败，丢盔解甲
